# Página ACA

Sitio web y sistema de gestión para la Asociación Civil Cetreros Argentinos.

## Estado

Proyecto en fase de especificación inicial (Spec-Driven Development).

## Alcance por etapas

1. Sitio institucional responsive.
2. Autenticación y área privada.
3. Gestión de socios y comisión directiva.
4. Panel administrativo.
5. Carnet virtual verificable mediante QR.
