# SPEC-001 — Sitio institucional responsive

- **Estado:** Próxima especificación a desarrollar
- **Versión:** 0.0.1

## Objetivo

Entregar la primera versión pública de Página ACA con navegación clara, identidad visual consistente y adaptación completa a móvil, tablet y escritorio.

## Secciones

- Header y navegación.
- Hero principal.
- Presentación de ACA.
- Introducción a la cetrería.
- Actividades destacadas.
- Beneficios o motivos para asociarse.
- Llamado a la acción.
- Footer y contacto.

## Contenido provisional

Hasta recibir material oficial, todo texto institucional específico y toda fotografía se tratarán como contenido provisional. No se inventarán autoridades, cifras, domicilio, redes ni requisitos de asociación.

## Criterios iniciales

- Diseño mobile-first.
- Menú móvil accesible.
- Componentes reutilizables.
- Sin datos personales reales.
- Buen contraste y foco visible.
- Imágenes optimizadas y con texto alternativo.
- Configuración central para enlaces y datos institucionales.
