# Roadmap inicial

## Hito 0 — Fundaciones

- Especificación maestra.
- Repositorio y convenciones.
- ADR tecnológicos.
- Identidad y contenido pendientes.

## Hito 1 — Web institucional

- Layout responsive.
- Navegación pública.
- Página de inicio.
- Secciones institucionales.
- Contacto y asociarse.
- SEO, accesibilidad y rendimiento inicial.

## Hito 2 — Identidad y socios

- Autenticación.
- Roles.
- Modelo de socios.
- Área privada inicial.

## Hito 3 — Administración

- ABM de socios con baja lógica.
- Comisión directiva.
- Auditoría.
- Filtros y exportaciones autorizadas.

## Hito 4 — Carnet virtual

- Emisión.
- QR.
- Verificación pública.
- Revocación y reemisión.

## Hito 5 — Evolución

- Cuotas, pagos, eventos y otros módulos, únicamente tras nuevas especificaciones.
